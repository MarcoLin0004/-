# bazi-local (本地运行)

一个完全本地运行的“生辰八字”页面，支持：
- 公历 / 农历输入
- 农历闰月（通过勾选“闰月”）
- 时辰可选：若未知时辰，会自动列出 12 个时辰对应的时柱可能性
- 输出含：八字/五行/纳音/十神/藏干/旬空/地势/命宫身宫/九星/大运等

> 仅供传统文化娱乐参考。

## 启动
```bash
npm i
npm run dev
```
打开终端提示的地址（默认 http://localhost:5173）

## 说明
- 本项目使用 `lunar-typescript`（6tail 系列）进行历法/八字计算。
- 农历闰月在库中通过“负数月份”表示：例如 闰六月 => month = -6。
